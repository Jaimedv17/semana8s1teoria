Basicamente es una especificacion de como resolver una clase de problema.

2.Datos de entrada / Datos de salida

Datos de entrada son los datos simples y compuestos presentados en clase o tambien en otras palabras pueden ser las erramientas o productos que se necesitan para poder aplicar el algoritmo. en cuanto a loda datos de salida, son el resultado entre el algoritmo y el dato de entrada.

Es un lenguaje de programacion pilar en todos los ambitos de programacion, sirve para volver interactivo a una pagina web, mostrar animaciones y mejorar la experiencia del los usuarios "orientado a objetos multiparadigma.
